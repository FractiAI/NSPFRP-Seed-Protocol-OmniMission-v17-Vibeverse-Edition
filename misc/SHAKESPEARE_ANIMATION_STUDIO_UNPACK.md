## Usage Examples

### Create Animation from Text

```nspfrp
mission CreateAnimationFromText {
  text: 'To be or not to be, that is the question';
  octave: TRANSCENDENCE;
  
  // Natural animation creation
  animation := createAnimation({
    text: text,
    analysis: shakespeare.analyze(text),
    characters: shakespeare.extractCharacters(text),
    narrative: shakespeare.structureNarrative(text),
    animation: animation.animate({
      characters: characters,
      scenes: narrative.scenes,
      style: 'shakespearean'
    })
  });
  
  // Protocol generation
  protocol := generateProtocol(animation);
  
  // Snapshot creation
  snapshot(protocol);
  
  return animation;
}
```

### Analyze Shakespearean Work

```nspfrp
mission AnalyzeShakespeareanWork {
  work: 'Hamlet';
  octave: SYMPHONY;
  
  // Natural analysis
  analysis := shakespeare.analyze({
    work: work,
    characters: extract(),
    themes: identify(),
    structure: analyze(),
    language: examine()
  });
  
  // Hero Host enrichment
  enriched := heroHost.enrich(analysis, 'william-shakespeare');
  
  // Protocol generation
  protocol := generateProtocol(enriched);
  
  return analysis;
}
```
